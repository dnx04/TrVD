**Check Context7 for library docs before using APIs.** When working with any library, framework, SDK, or API — even well-known ones like tree-sitter, PyTorch, or gensim — use the `mcp__plugin_context7_context7__resolve-library-id` + `mcp__plugin_context7_context7__query-docs` tools to fetch current documentation rather than relying on training data. This applies to API syntax, configuration, version migration, and CLI tool usage.
